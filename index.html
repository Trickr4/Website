Software and Tools

Stacked Deck

Software developed in Unity(C#) for the Riot Games's Legends of Runeterra Developer Challenge 2019. The Developer challenge goal is to make a software tool for their game Legends of Runeterra. Stacked Deck combined random card under the constraints to make a deck. This tool allowed player more randomness to their gameplay and fun, while allowing them to save and import the deck through MongoDB and the game's api.

One Thread

A simple android application inspired by reddit that uses a thread like messanger board. The software utilizes Google Firebase to store data from users and their post to authenticate and display correct information in the app.
